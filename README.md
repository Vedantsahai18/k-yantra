# Installation

Clone the repository
```
git clone https://github.com/Vedantsahai18/k-yantra
```

## Run UI on local
Setup Python environment
```
python -m venv venv

[Windows users]
.\venv\Scripts\activate

[Ubuntu]
source venv/bin/activate

pip install -r requirements.txt
```

## Run everything 
```
python app.py
```
